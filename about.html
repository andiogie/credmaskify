Tentang Credential Masker

Credential Masker adalah tool gratis untuk menyamarkan informasi sensitif — IP address, port, hostname/domain, email, dan pola password/token — dari teks apa pun sebelum kamu bagikan ke orang lain, misalnya lewat screenshot, chat support, tiket bug report, atau dokumentasi internal.

Kenapa Tool Ini Dibuat

Sering kali kita perlu share hasil terminal, log aplikasi, atau file konfigurasi ke orang lain untuk minta bantuan debug — tapi di dalamnya ada IP internal, hostname server produksi, atau bahkan password yang ke-copy tanpa sengaja. Menyamarkan itu manual satu-satu itu makan waktu dan gampang kelewatan. Credential Masker otomatiskan bagian itu, sambil tetap kasih kamu kontrol penuh untuk edit manual hasilnya.

Kenapa Berjalan di Browser (Client-Side)

Semua proses masking terjadi 100% di browser kamu sendiri menggunakan JavaScript murni. Teks yang kamu tempel tidak pernah dikirim ke server mana pun — jadi kamu tetap bisa pakai tool ini untuk data yang sensitif tanpa khawatir soal kebocoran lewat pihak ketiga.

Cara Kerja Singkat

Tool ini mencocokkan pola teks lewat regular expression (IP address, port setelah titik dua atau flag -p, hostname/domain, email, serta pasangan key=value seperti password= atau token=), lalu mengganti setiap huruf/angka yang cocok dengan karakter x — panjang dan tanda baca aslinya tetap dipertahankan. Hasilnya tetap bisa kamu edit manual di kotak output kapan saja.
